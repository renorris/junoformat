JunoFormatBeta 1.1

Fixes formatting issues when copying text from Google Docs to Juno Docs.

About

This tool formats text from Google Docs so Juno Docs can display it properly. Changelog

Implemented Features:
  • Fixes bold, italic, and underlined text
  • Inserts line breaks when needed
  • Applies appropriate line indentations
  • Formats bullet points, lettering, numbering, and indentations on ordered and unordered lists
Planned Features:
  • Hyperlinks
  • Images (Currently you must manually insert images from within the Juno editor)
Please note:
  • This project is incomplete and can be unreliable.
  • If you're using this for something important, please double-check that all your text has pasted properly in Juno.
  • Everything you input into this site stays on your computer and isn't sent anywhere on the internet. This includes the text you paste.

If you receive errors when attempting to use the tool: Refresh this page, re-copy your Google Docs text, then paste it again here. If that doesn't help, feel free to email the address marked below with your issue.

For any questions/suggestions please email junoformat@gmail.com

What's New

Version 1.1 introduces proper formatting for bulleted and numbered lists.

What's included:
  • List indentation
  • Multiple bullet icon types
  • Alphabetical lists - E.g.
    a. Lorem ipsum dolor sit amet
    b. Consectetur adipiscing elit
  • Roman numeral lists - E.g.
    i. Ed do eiusmod tempor
    ii. Incididunt ut labore
